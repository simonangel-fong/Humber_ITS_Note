# Command: `man`

`man`: A Linux command to read the manual of any other command or utilities.

- Syntax: `$man <command_name>`

- 实践：离开手册press `Q` to leave

- i.e.:

```java

$man man // to check manual of the man command

$man touch

$man cat

```
