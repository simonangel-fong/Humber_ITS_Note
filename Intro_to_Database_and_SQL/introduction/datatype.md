## Display 显示

- Alignment <br>不同的的数据类型有不同的对齐
   - Left-aligned 左对齐
       - text / character fields
        - date fields
    - Right-aligned 右对齐
        - numeric data

- Insignificant zeroes
    - By default, Oracle does not display insignificant zeroes.<br> i.e.: `54.50` => `54.5`
